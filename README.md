# Valentine Khalisa (React)

## Run locally

```bash
npm install
npm run dev
```

## Build for Vercel

```bash
npm run build
```

Vercel settings:
- Framework Preset: Vite
- Build Command: npm run build
- Output Directory: dist

## Add photos

Put photos in `public/photos` with names:
- `khalisa-1.jpg`
- `khalisa-2.jpg`
- `khalisa-3.jpg`
- `khalisa-4.jpg`
